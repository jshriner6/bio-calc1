Activity 3.1.4. Quadratic Approximation.

Let \(f(x)\) be a twice differentiable function and \(1\) the center we would like to use for an approximation. Suppose \(Q(x) = d + c(x-1) + b(x-1)^2\) is the best quadratic approximation, meaning
  • \(Q(1) = f(1)\text{,}\)
  • \(Q'(1) = f'(1)\text{,}\) and
  • \(Q''(1) = f''(1)\text{.}\)
  1. Determine the value \(Q(1)\text{.}\) What does this tell you about the value of the parameter \(d\text{?}\)
  2. Compute \(Q'(x)\) and determine the value \(Q'(1)\text{.}\) What does this tell you about the value of the parameter \(c\text{?}\)
  3. Compute \(Q''(x)\) and determine the value \(Q''(1)\text{.}\) What does this tell you about the value of the parameter \(b\text{?}\)
  4. Write down the general formula for \(Q(x)\) in terms of the function \(f\) and its derivatives. How does it compare to the linear approximation \(L(x)\) of \(f\) centered at \(1\text{?}\)
