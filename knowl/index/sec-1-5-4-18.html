Another common base for writing exponential functions is the number \(e\text{.}\) This is an irrational number equal to approximately \(2.718\text{,}\) which is special in calculus for reasons we will see in SectionÂ 2.5. Writing an exponential function such as \(2^x\) using base \(e\) means re-writing \(2^x\) as \(e^{kx}\) for some constant \(k\text{.}\) Using rules of exponents to write
\begin{equation*} 2^x = e^{kx} = (e^k)^x\text{,} \end{equation*}
we see this means finding a constant \(k\) such that \(2 = e^k\text{.}\) How do we find such a \(k\text{?}\)
