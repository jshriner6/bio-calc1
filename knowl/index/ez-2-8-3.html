Exercise 3.

Recall we say that function \(g\) dominates function \(f\) provided that \(\lim_{x \to \infty} f(x) = \infty\text{,}\) \(\lim_{x \to \infty} g(x) = \infty\text{,}\) and \(\lim_{x \to \infty} \frac{f(x)}{g(x)} = 0\text{.}\) Use limits to justify each of your answers to the questions below.
  1. Which function dominates the other: \(x^3\) or \(x^2\text{?}\)
  2. Which function dominates the other: \(2^x\) or \(5^x\text{?}\)
  3. Which function dominates the other: \(\ln(x)\) or \(\sqrt{x}\text{?}\)
  4. Which function dominates the other: \(\ln(x)\) or \(\sqrt[n]{x}\text{?}\) (\(n\) can be any positive integer)
  5. Explain why \(e^x\) will dominate any polynomial function.
  6. Explain why \(x^n\) will dominate \(\ln(x)\) for any positive integer \(n\text{.}\)
  7. Give any example of two nonlinear functions such that neither dominates the other.
Answer.
  1. \(\displaystyle x^3\)
  2. \(\displaystyle 5^x\)
  3. \(\displaystyle \sqrt{x}\)
  4. \(\displaystyle \sqrt[n]{x}\)
